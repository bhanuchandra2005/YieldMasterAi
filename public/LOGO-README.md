# App logo

To use your own logo (e.g. the YieldMasterAI logo with plants + circuit):

1. **In-app logo (navbar, login, footer, etc.)**  
   Save your image as **`logo.png`** in this folder (`public/`).  
   Recommended: square or horizontal, at least 128×128 px.  
   The app will show it everywhere; if the file is missing, a leaf icon is shown instead.

2. **Browser tab icon (favicon)**  
   To use the same logo in the tab, replace **`favicon.svg`** in this folder with your logo (as SVG), or add **`favicon.png`** (e.g. 32×32) and the app can be updated to use it.
